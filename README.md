### ▰ Full-Stack & Mobile Architecture
> ▪ Repositories containing the Kotlin and Jetpack Compose source code for offline-native Android hardware access (Yantra)  
> ▪ Next.js, Supabase, and Vercel deployment architectures for web platforms (Ignis, Narmada School Website)  
> ▪ Implementation of agentic AI workflows and API routing (Gemini API integrations)  

### ▰ Embedded Systems & Bare-Metal Firmware
> ▪ C++ codebase for ESP32 microcontrollers  
> ▪ Custom drivers and scripts for raw sensor telemetry extraction (MPU6050, smartphone accelerometers/gyroscopes)  
> ▪ Code optimized to compile and execute under severe compute limitations  

### ▰ Physics & Materials Science Data Pipelines
> ▪ Mathematical models and data-logging scripts used to measure RF attenuation for the carbonized RAM tiles  
> ▪ Kinematic equations and spatial modeling scripts for the APEX Frame exosuit  
> ▪ LaTeX source code and mathematical derivations for the independent theoretical physics paper
